# dotfiles
My personal dotfiles managed with [dotbot](https://github.com/anishathalye/dotbot).

## Requirements
- [zsh](http://www.zsh.org/)
- [git](http://www.git-scm.com/)
- [vim](http://www.vim.org/)

## Installation
```shell
$ cd ~
$ git clone https://github.com/aspyatkin/dotfiles
$ cd dotfiles
$ ./install
```
